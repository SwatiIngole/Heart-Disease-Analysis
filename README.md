❤️ Heart Disease Analysis Dashboard

📌 Project Overview

The Heart Disease Analysis Dashboard is a Power BI project designed to analyze patient health data and identify key risk factors contributing to heart disease.

This dashboard provides interactive visual insights into patient demographics, medical conditions, and lifestyle factors to support data-driven healthcare decisions.

🎯 Objective

The main objective of this project is to:

Analyze heart disease patient data

Identify high-risk groups

Understand major contributing health factors

Provide meaningful visual insights using Power BI

📊 Dashboard Features

✅ Total Patients & Heart Disease Count Overview

✅ Gender-wise and Age-wise Distribution

✅ Analysis of Chest Pain Type

✅ Blood Pressure & Cholesterol Level Insights

✅ Maximum Heart Rate Analysis

✅ Exercise-Induced Angina Analysis

✅ Interactive Filters & Slicers for Dynamic Analysis

📂 Dataset Information

The dataset includes the following attributes:

Age

Gender

Chest Pain Type

Resting Blood Pressure

Cholesterol Level

Fasting Blood Sugar

Rest ECG

Maximum Heart Rate Achieved

Exercise-Induced Angina

ST Depression (Oldpeak)

Target (Heart Disease Presence)

🛠 Tools & Technologies Used

Power BI – Data Visualization & Dashboard Creation

Power Query – Data Cleaning & Transformation

DAX (Data Analysis Expressions) – Calculated Measures & KPIs

🔍 Key Insights

Higher risk observed in specific age groups.

Certain chest pain types are strongly associated with heart disease.

Elevated cholesterol and blood pressure levels increase risk probability.

Exercise-induced angina shows significant correlation with heart disease presence.

📈 Business Impact

Helps in identifying high-risk patients.

Supports healthcare professionals in preventive decision-making.

Enables better understanding of heart disease patterns through visualization.

🚀 How to Use

Download the .pbix file from this repository.

Use slicers and filters to interact with the dashboard.

Explore insights dynamically.
